Notebooks usados em sala de aula na disciplina Linguagem de Programação I.

[![Binder](https://mybinder.org/badge.svg)](https://mybinder.org/v2/gh/catolicasc-joinville/lp1-notebooks/master)

## Instalando

```sh
pip install pipenv
pipenv install
```

## Executar

```sh
pipenv run jupyter lab
```

ou

```sh
docker-compose up
```
